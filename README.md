# Vehicle Hub

Vehicle Hub is a web application where users can log in using their Gmail accounts. Users can add, book, and view vehicles. There is also a **My Vehicles** page that shows all the vehicles added by the user.  

## Features

<ul>
  <li>Gmail authentication for users</li>
  <li>Add new vehicles</li>
  <li>Book vehicles</li>
  <li>View all vehicles added by the user (My Vehicles)</li>
</ul>

## Technologies Used

<ul>
  <li>Frontend: React</li>
  <li>Backend: Node.js, Express.js</li>
  <li>Database: MongoDB</li>
</ul>
